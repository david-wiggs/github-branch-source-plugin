The URL of the external authentication service that will receive credentials and return GitHub tokens.

Request Format: The service should accept POST requests with JSON body containing:
  • repositoryUrl - The GitHub repository URL being accessed
  • username - The username from the Jenkins credentials
  • password - The password from the Jenkins credentials
  • apiUri - The GitHub API URI (e.g., https://api.github.com)
  • repositoryOwner - The repository owner/organization
  • repositoryName - The repository name

Response Format: The service should return JSON containing:
  • success - Boolean indicating if authentication succeeded
  • token - The GitHub token to use for API access
  • message - Optional error message if authentication failed

Example: https://auth.mycompany.com/github-token
